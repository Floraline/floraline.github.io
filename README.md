# Site Saint-Valentin

Site web thématique créé pour la Saint-Valentin, 
avec une expérience utilisateur en deux étapes.

## Fonctionnalités

- Page d'accès protégée par mot de passe personnalisé
- Message d'erreur dynamique selon l'identifiant saisi
- Page principale avec contenu de connexion
- Page secondaire avec contenu visuel thématique (lettre)
- Musique d'ambiance
- Formulaire interactif

## Langages

HTML · CSS · JavaScript
